3 ## NCEPLIBS-ncio
4 
5 NetCDF read/write modules for the NCEP models. This is part of the
6 [NCEPLIBS](https://github.com/NOAA-EMC/NCEPLIBS) project. The NCIO module
7 is mainly used by GSI as netcdf I/O interface.
8 
9 The NCEPLIBS-ncio code here: https://github.com/NOAA-EMC/NCEPLIBS-ncio.
