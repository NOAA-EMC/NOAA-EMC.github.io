7 C ABSTRACT: CONVERT A NORTHERN HEMISPHERE 1.0 DEGREE LAT.,LON. 361 BY

22 C INTERP - 1 LINEAR INTERPOLATION , NE.1 BIQUADRATIC
23 C
24 C INPUT FILES: NONE

143  xdelj(k) = w2(k) - jv(k)
144  ip1(k) = iv(k) + 1
145  jy(k,3) = jv(k) + 1
