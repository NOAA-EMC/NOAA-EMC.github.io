7 C SUBPROGRAM: SPTRANFV PERFORM A VECTOR SPHERICAL TRANSFORM
8 C PRGMMR: IREDELL ORG: W/NMC23 DATE: 96-02-29
9 C
10 C ABSTRACT: THIS SUBPROGRAM PERFORMS A SPHERICAL TRANSFORM
11 C BETWEEN SPECTRAL COEFFICIENTS OF DIVERGENCES AND CURLS
12 C AND VECTOR FIELDS ON A GLOBAL CYLINDRICAL GRID.
13 C THE WAVE-SPACE CAN BE EITHER TRIANGULAR OR RHOMBOIDAL.
14 C THE GRID-SPACE CAN BE EITHER AN EQUALLY-SPACED GRID
15 C (WITH OR WITHOUT POLE POINTS) OR A GAUSSIAN GRID.
16 C THE WAVE AND GRID FIELDS MAY HAVE GENERAL INDEXING,
17 C BUT EACH WAVE FIELD IS IN SEQUENTIAL 'IBM ORDER',
18 C I.E. WITH ZONAL WAVENUMBER AS THE SLOWER INDEX.
19 C TRANSFORMS ARE DONE IN LATITUDE PAIRS FOR EFFICIENCY;
20 C THUS GRID ARRAYS FOR EACH HEMISPHERE MUST BE PASSED.
21 C IF SO REQUESTED, JUST A SUBSET OF THE LATITUDE PAIRS
22 C MAY BE TRANSFORMED IN EACH INVOCATION OF THE SUBPROGRAM.
23 C THE TRANSFORMS ARE ALL MULTIPROCESSED OVER LATITUDE EXCEPT
24 C THE TRANSFORM FROM FOURIER TO SPECTRAL IS MULTIPROCESSED
25 C OVER ZONAL WAVENUMBER TO ENSURE REPRODUCIBILITY.
26 C TRANSFORM SEVERAL FIELDS AT A TIME TO IMPROVE VECTORIZATION.
27 C SUBPROGRAM CAN BE CALLED FROM A MULTIPROCESSING ENVIRONMENT.

29 C PROGRAM HISTORY LOG:
