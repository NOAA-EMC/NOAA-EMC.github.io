5 C$$$ SUBPROGRAM DOCUMENTATION BLOCK
6 C
7 C SUBPROGRAM: SPTGPMV TRANSFORM SPECTRAL VECTOR TO MERCATOR
8 C PRGMMR: IREDELL ORG: W/NMC23 DATE: 96-02-29
9 C
10 C ABSTRACT: THIS SUBPROGRAM PERFORMS A SPHERICAL TRANSFORM
11 C FROM SPECTRAL COEFFICIENTS OF DIVERGENCES AND CURLS
12 C TO VECTOR FIELDS ON A MERCATOR GRID.
13 C THE WAVE-SPACE CAN BE EITHER TRIANGULAR OR RHOMBOIDAL.
14 C THE WAVE AND GRID FIELDS MAY HAVE GENERAL INDEXING,
15 C BUT EACH WAVE FIELD IS IN SEQUENTIAL 'IBM ORDER',
16 C I.E. WITH ZONAL WAVENUMBER AS THE SLOWER INDEX.
17 C THE MERCATOR GRID IS IDENTIFIED BY THE LOCATION
18 C OF ITS FIRST POINT AND BY ITS RESPECTIVE INCREMENTS.
19 C THE TRANSFORMS ARE ALL MULTIPROCESSED OVER SECTOR POINTS.
20 C TRANSFORM SEVERAL FIELDS AT A TIME TO IMPROVE VECTORIZATION.
21 C SUBPROGRAM CAN BE CALLED FROM A MULTIPROCESSING ENVIRONMENT.
22 C
23 C PROGRAM HISTORY LOG:
24 C 96-02-29 IREDELL
25 C 1998-12-15 IREDELL OPENMP DIRECTIVES INSERTED
26 C
27 C USAGE: CALL SPTGPMV(IROMB,MAXWV,KMAX,MI,MJ,
28 C & KWSKIP,KGSKIP,NISKIP,NJSKIP,
29 C & RLAT1,RLON1,DLAT,DLON,WAVED,WAVEZ,UM,VM)
30 C INPUT ARGUMENTS:
31 C IROMB - INTEGER SPECTRAL DOMAIN SHAPE
32 C (0 FOR TRIANGULAR, 1 FOR RHOMBOIDAL)
33 C MAXWV - INTEGER SPECTRAL TRUNCATION
34 C KMAX - INTEGER NUMBER OF FIELDS TO TRANSFORM.
35 C MI - INTEGER NUMBER OF POINTS IN THE FASTER ZONAL DIRECTION
36 C MJ - INTEGER NUMBER OF POINTS IN THE SLOWER MERID DIRECTION
37 C KWSKIP - INTEGER SKIP NUMBER BETWEEN WAVE FIELDS
38 C (DEFAULTS TO (MAXWV+1)*((IROMB+1)*MAXWV+2) IF KWSKIP=0)
39 C KGSKIP - INTEGER SKIP NUMBER BETWEEN GRID FIELDS
40 C (DEFAULTS TO MI*MJ IF KGSKIP=0)
41 C NISKIP - INTEGER SKIP NUMBER BETWEEN GRID I-POINTS
42 C (DEFAULTS TO 1 IF NISKIP=0)
43 C NJSKIP - INTEGER SKIP NUMBER BETWEEN GRID J-POINTS
44 C (DEFAULTS TO MI IF NJSKIP=0)
45 C RLAT1 - REAL LATITUDE OF THE FIRST GRID POINT IN DEGREES
46 C RLON1 - REAL LONGITUDE OF THE FIRST GRID POINT IN DEGREES
47 C DLAT - REAL LATITUDE INCREMENT IN DEGREES SUCH THAT
48 C D(PHI)/D(J)=DLAT*COS(PHI) WHERE J IS MERIDIONAL INDEX.
49 C DLAT IS NEGATIVE FOR GRIDS INDEXED SOUTHWARD.
50 C (IN TERMS OF GRID INCREMENT DY VALID AT LATITUDE RLATI,
51 C THE LATITUDE INCREMENT DLAT IS DETERMINED AS
52 C DLAT=DPR*DY/(RERTH*COS(RLATI/DPR))
53 C WHERE DPR=180/PI AND RERTH IS EARTH'S RADIUS)
54 C DLON - REAL LONGITUDE INCREMENT IN DEGREES SUCH THAT
55 C D(LAMBDA)/D(I)=DLON WHERE I IS ZONAL INDEX.
56 C DLON IS NEGATIVE FOR GRIDS INDEXED WESTWARD.
57 C WAVED - REAL (*) WAVE DIVERGENCE FIELDS
58 C WAVEZ - REAL (*) WAVE VORTICITY FIELDS
59 C OUTPUT ARGUMENTS:
60 C UM - REAL (*) MERCATOR U-WINDS
61 C VM - REAL (*) MERCATOR V-WINDS
62 C
63 C SUBPROGRAMS CALLED:
64 C SPWGET GET WAVE-SPACE CONSTANTS
65 C SPLEGEND COMPUTE LEGENDRE POLYNOMIALS
66 C SPSYNTH SYNTHESIZE FOURIER FROM SPECTRAL
67 C SPDZ2UV COMPUTE WINDS FROM DIVERGENCE AND VORTICITY
68 C
69 C ATTRIBUTES:
70 C LANGUAGE: FORTRAN 77
71 C
72 C$$$

148  ENDDO
149  ENDDO
