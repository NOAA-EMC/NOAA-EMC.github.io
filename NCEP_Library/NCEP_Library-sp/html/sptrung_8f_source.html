6 C
7 C SUBPROGRAM: SPTRUNG SPECTRALLY INTERPOLATE SCALARS TO STATIONS
8 C PRGMMR: IREDELL ORG: W/NMC23 DATE: 96-02-29
9 C
10 C ABSTRACT: THIS SUBPROGRAM SPECTRALLY TRUNCATES SCALAR FIELDS
11 C ON A GLOBAL CYLINDRICAL GRID, RETURNING THE FIELDS
12 C TO SPECIFIED SETS OF STATION POINTS ON THE GLOBE.
13 C THE WAVE-SPACE CAN BE EITHER TRIANGULAR OR RHOMBOIDAL.
14 C THE GRID-SPACE CAN BE EITHER AN EQUALLY-SPACED GRID
15 C (WITH OR WITHOUT POLE POINTS) OR A GAUSSIAN GRID.
16 C THE GRID AND POINT FIELDS MAY HAVE GENERAL INDEXING.
17 C THE TRANSFORMS ARE ALL MULTIPROCESSED.
18 C TRANSFORM SEVERAL FIELDS AT A TIME TO IMPROVE VECTORIZATION.
19 C SUBPROGRAM CAN BE CALLED FROM A MULTIPROCESSING ENVIRONMENT.
20 C
21 C PROGRAM HISTORY LOG:
22 C 96-02-29 IREDELL
23 C
24 C USAGE: CALL SPTRUNG(IROMB,MAXWV,IDRTI,IMAXI,JMAXI,KMAX,NMAX,
25 C & IPRIME,ISKIPI,JSKIPI,KSKIPI,KGSKIP,
26 C & NRSKIP,NGSKIP,JCPU,RLAT,RLON,GRIDI,GP)
27 C INPUT ARGUMENTS:
28 C IROMB - INTEGER SPECTRAL DOMAIN SHAPE
29 C (0 FOR TRIANGULAR, 1 FOR RHOMBOIDAL)
30 C MAXWV - INTEGER SPECTRAL TRUNCATION
31 C IDRTI - INTEGER INPUT GRID IDENTIFIER
32 C (IDRTI=4 FOR GAUSSIAN GRID,
33 C IDRTI=0 FOR EQUALLY-SPACED GRID INCLUDING POLES,
34 C IDRTI=256 FOR EQUALLY-SPACED GRID EXCLUDING POLES)
35 C IMAXI - INTEGER EVEN NUMBER OF INPUT LONGITUDES.
36 C JMAXI - INTEGER NUMBER OF INPUT LATITUDES.
37 C KMAX - INTEGER NUMBER OF FIELDS TO TRANSFORM.
38 C NMAX - INTEGER NUMBER OF STATION POINTS TO RETURN
39 C IPRIME - INTEGER INPUT LONGITUDE INDEX FOR THE PRIME MERIDIAN.
40 C (DEFAULTS TO 1 IF IPRIME=0)
41 C (OUTPUT LONGITUDE INDEX FOR PRIME MERIDIAN ASSUMED 1.)
42 C ISKIPI - INTEGER SKIP NUMBER BETWEEN INPUT LONGITUDES
43 C (DEFAULTS TO 1 IF ISKIPI=0)
44 C JSKIPI - INTEGER SKIP NUMBER BETWEEN INPUT LATITUDES FROM SOUTH

54 C RLAT - REAL (*) STATION LATITUDES IN DEGREES
55 C RLON - REAL (*) STATION LONGITUDES IN DEGREES
56 C JCPU - INTEGER NUMBER OF CPUS OVER WHICH TO MULTIPROCESS
57 C (DEFAULTS TO ENVIRONMENT NCPUS IF JCPU=0)
58 C GRIDI - REAL (*) INPUT GRID FIELDS
59 C OUTPUT ARGUMENTS:
60 C GP - REAL (*) STATION POINT SETS
61 C
62 C SUBPROGRAMS CALLED:
63 C SPTRAN PERFORM A SCALAR SPHERICAL TRANSFORM
64 C SPTGPT TRANSFORM SPECTRAL SCALAR TO STATION POINTS
65 C NCPUS GETS ENVIRONMENT NUMBER OF CPUS
66 C
67 C REMARKS: MINIMUM GRID DIMENSIONS FOR UNALIASED TRANSFORMS TO SPECTRAL:
